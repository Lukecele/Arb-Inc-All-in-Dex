# Arbitrage Inception — Technical Whitepaper

## Abstract
Arbitrage Inception is a permissionless, open-source DeFi frontend interface
on BNB Smart Chain. This document describes the protocol mechanics, 
revenue flows, and governance model.

## Token: ARB Inc
- Contract: 0x5ee54869ecd5e752c31af095187326d4a4d50e1c (BSC)
- Ownership: RENOUNCED (immutable, no admin keys)
- Total Supply: 1,000,000,000 ARB
- Tax: 4% on buy/sell → 100% routed to treasury

## Revenue Streams
1. Token Tax (4%): All buy/sell transactions → Treasury
2. DEX Fee (0.1%): KyberSwap widget integration → Treasury  
3. Bridge Referral: Mayan Finance referral → Treasury

## Treasury Distribution
- Treasury wallet: 0x66BB01F14229E2179bAD84D52A69C0e4628dE63f
- Distribution: 80% to ARB holders / 20% to protocol development fund
- Frequency: Every 6 hours, automated
