# Arb Inc All-in-Dex

Open-source, community-driven decentralized exchange aggregator on BNB Smart Chain.

**License:** MIT  
**Source:** https://github.com/Lukecele/Arb-Inc-All-in-Dex  
**Community:** https://t.me/ArbitrageInception

## Description

Arbitrage Inception is a permissionless, open-source frontend interface for
interacting with decentralized liquidity on BNB Smart Chain. It is not a
custodian, broker, financial institution, or Virtual Asset Service Provider (VASP)
under Reg. EU 2023/1114 (MiCA). All trades are executed directly on-chain via
the user's own wallet. There is no team, no company, and no identifiable developer
behind this project — it is autonomous open-source software.

## Tokenomics

- **20% burn on every distribution** — tokens permanently removed from circulation
- **50% of DEX revenue** converted to BNB and sent to distribution wallet
- **Every 12 hours:**
  - **40% BNB rewards** distributed to holders
  - **20% buyback and burn** forever

## Key Features

- Sustainable, deflationary reward system
- Automatic buyback and burn mechanism
- Fully open-source frontend (MIT License)
- Transparent on-chain tokenomics
- No team, no admin keys, no custodians
