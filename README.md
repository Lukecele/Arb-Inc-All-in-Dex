# Arb Inc All-in-Dex

## Description
Arb Inc All-in-Dex is a decentralized exchange platform with innovative tokenomics designed to reward holders and ensure sustainable growth.

## Tokenomics

### Distribution Mechanism
- **20% burn on every distribution** - Tokens are permanently removed from circulation
- **50% of dex revenue** will be converted to BnB and sent to distribution wallet
- **Every 12 hours:**
  - **40% BnB rewards** distributed to holders and dev
  - **20% buyback and burn** forever

This ensures a deflationary model while rewarding long-term holders.

### Key Features
- Sustainable reward system
- Automatic buyback and burn mechanism
- Community-focused development
- Transparent tokenomics
